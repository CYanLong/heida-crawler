
###基于 `Python` 爬虫的微信公众号。
---

`python3`, `flask`,  `requests`, `scrapy` 和 某云服务器搭建。

`flask`: 轻量级的web框架。主要用来接收来自微信用户的访问并进行回复。

`requests`: 简洁好用的 `HTTP` 框架。用来请求互联网信息。

`scrapy`: 爬虫框架。很方便的获取到页面的信息。

---
###features

1.黑大图书馆藏信息的检索<br/>

2.获取笑话

3.电影推荐功能

4.电影信息查询功能
